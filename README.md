### 📌 Project Overview
This repository serves as a comprehensive archive of practical coding implementations derived from the Google Advanced Data Analytics professional track. It documents the hands-on journey from data cleaning to advanced machine learning modeling, focusing on end-to-end data analytics and predictive modeling workflows.

The codebase spans key data science domains, including:
* **Exploratory Data Analysis (EDA):** Data wrangling, feature engineering, and visual insights.
* **Statistical Inference & Hypothesis Testing:** Evaluating distributions and experimental validation.
* **Regression Modeling:** Building and interpreting Linear and Logistic Regression models.
* **Machine Learning Pipelines:** Developing, tuning, and evaluating Tree-based models (Random Forest, XGBoost) and classification frameworks.

### 📌 项目概述（Project Overview）
本仓库为 Google 高级数据分析实战练习的完整归档，记录了从数据清洗到高级机器学习建模的全流程 Python 实现，旨在展示端到端的数据分析与预测建模能力。

包含的代码模块涵盖以下数据科学核心领域：
* **探索性数据分析 (EDA)：** 数据清洗、特征工程与数据可视化洞察。
* **统计推断与假设检验：** 数据分布检验、置信区间计算与 A/B 测试/假设验证。
* **回归分析模型：** 线性回归（Linear Regression）与逻辑回归（Logistic Regression）的构建与解读。
* **机器学习工作流：** 基于树模型（随机森林 Random Forest、XGBoost）的分类模型构建、超参数调优与评估。
